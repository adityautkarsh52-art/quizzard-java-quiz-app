# Quizzard – Java Quiz Application

Quizzard is a GUI-based quiz application developed using **Java Swing**.
The application allows users to register, log in, select quiz levels, answer questions, and view their final scores.

---

## Features

* User Registration and Login System
* Graphical User Interface using Java Swing
* Multiple Quiz Levels
* Timer-based MCQ Questions
* Score Calculation
* Result Display

---

## Technologies Used

* Java
* Java Swing
* File Handling

---

## Project Structure

login.java – Login window
register.java – User registration form
name.java – Player name input
level.java – Quiz level selection

Quiz1.java – Quiz level 1
Quiz2.java – Quiz level 2
Quiz3.java – Quiz level 3

Rules.java – Rules page
Rules1.java – Rules page for level 2
Rules2.java – Rules page for level 3

Score.java – Final score display

questionslast.txt – Question dataset
questionslast1.txt – Question dataset
questionslast2.txt – Question dataset

---

## How to Run

Open terminal in the project folder.

Compile the program:

javac *.java

Run the application:

java login
